# Flavors of India

**Website:** [https://flavorsofindiakh.org](https://flavorsofindiakh.org)

---

## Description

Flavors of India is a modern restaurant website showcasing authentic Indian and Nepali cuisine.  
It highlights signature dishes like Butter Chicken, Pani Puri, Hyderabadi Dum Biryani, Chicken Jhol Momo, Nepali Special Thali, and Murgh Handi.  

The site provides a **responsive design**, smooth animations, and **SEO-optimized content** for better online visibility.

---

## Features

- Fully **responsive layout** for desktop, tablet, and mobile  
- Modern **Tailwind CSS** design with smooth hover animations  
- **Next.js Image optimization** for fast-loading images  
- Menu categories with **images, descriptions, and prices**  
- SEO-friendly meta tags, alt attributes, and structured data  
- Contact information and social media integration (Facebook, Instagram, TikTok)  

---

## Tech Stack

- **Framework:** Next.js (React)  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Image Hosting:** Cloudinary  
- **Deployment:** Vercel
